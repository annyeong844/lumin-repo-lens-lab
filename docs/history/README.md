# History Staging Area

This directory is the intended landing zone for phase history,
retrospectives, and design-session notes that should remain in the
repository without continuing to crowd the shipping surface.

For the broader docs map, start at [`docs/README.md`](../README.md).

Phase session notes now live under:

- `docs/history/phases/`

Typical move targets:

- `docs/history/phases/p1/` through `docs/history/phases/p6/`
- `docs/history/FP-41-regression.md`
- false-positive retrospectives
- closed session notes
- benchmark writeups and postmortems

Maintainer-facing living specs now belong under:

- `docs/spec/`

This area is documentation history, not the public capability contract.
The public contract remains anchored at:

- `SKILL.md`
- `README.md`
- `audit-repo.mjs`
- `canonical/`
